# Public-Safe Data Policy

Service Pricing Calculator is designed for public demos, forks, screenshots, and code review. Keep all examples fictional unless the material is explicitly approved for public release.

## Allowed

- Synthetic people, companies, projects, tickets, contacts, dates, IDs, and budgets.
- Public documentation links and public GitHub links.
- Screenshots generated from this repository's fictional sample data.
- Example exports that can be shared in an issue or pull request.

## Not Allowed

- Real customer records, contacts, exports, contracts, analytics, payment details, or support tickets.
- Private URLs, private workspace screenshots, credentials, or copied private client material.
- Data copied from production systems unless it has been rewritten as synthetic fixture data.

## Review Before Publishing

Check README examples, screenshots, JSON/CSV/Markdown exports, fixtures, and issue comments before sharing. When unsure, replace the detail with a synthetic equivalent.
